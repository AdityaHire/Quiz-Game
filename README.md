# Quiz Game

A fun and interactive web-based quiz game built with HTML, CSS, and JavaScript.

## Features

- **Random Questions**: Each game selects 5 random questions from a pool of 10
- **Interactive Interface**: Click-based answer selection with visual feedback
- **Real-time Scoring**: Track your score as you progress
- **Progress Tracking**: Visual progress bar and question counter
- **Performance Feedback**: Get personalized messages based on your score
- **Responsive Design**: Works on both desktop and mobile devices
- **Restart Functionality**: Play multiple rounds easily
- 
## How to Play

1. Open `index.html` in any web browser
2. Click "Start Quiz" to begin
3. Read each question and click your answer
4. Watch the color feedback (green = correct, red = wrong)
5. View your final score and performance message
6. Click "Restart Quiz" to play again
